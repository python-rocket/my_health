# MY HEALTH



## Features (pain point)
- Experts and their Content (e.g Youtube or Books or blogs)
- TestUploader (upload pdf -> save to database)
- TestingOverview (possible tests, mark done or not dont)
- AI Doctor (checks test result and prioritises fix of issues (e.g Candida, Gutflora, MTHFR))
- Cockpit Dashboard (shows e.g test results, action items, AI doctor)
- DoctorFinder (e.g doctors for IBS treatment, or more concrete Candidas)
- ProtocolCreator (have predefined protocols, create custom protocols)
- SymptomChecker (check and track symptoms over time. E.g full nose, or gluten allergy, or bad sleep..; shows how symptoms got better or worse across time)

## Sales
- Can be used by doctors for their patients (easy overview and combined interface for interaction)
- Can be used by private people to have a personal health overview
- Can be used for data analysis
- Can be used to connect people into "interest groups" -> e.g everyone with MTHFR
- Can have affilate to: Supplements, Doctors
- Can have a "protocol marketplace"
